Le but de ce fichier est de décrires les différentes étapes nécessaires pour effectuer une estimation de paramètres pour le modèle Ecomersitem

1) Le script réalisant l'analyse de sensibilité se nomme : estimation.R
2) Dans estimation.R, les actions suivantes sont à réaliser :
	- définir la variable PPath (ligne 5 du fichier) en spécifiant où se situe le fichier "ECOMERISTEM_PARAMETERS.txt"
	- définir la variable MPath (ligne 6 du fichier) en spécifiant où se situent les fichiers meteo
	- définir la variable VPath (ligne 7 du fichier) en spécifiant où se situe le fichier vobs
	- définir la variable VName (ligne 8 du fichier) en spécifiant le nom du fichier vobs (ajouter l'extension .txt)
	- définir la variable VECName (ligne 9 du fichier) en spécifiant le nom du fichier EC (ajouter l'extension .txt)*
	- définir la liste ParamOfInterest (ligne 10 du fichier) en spécifiant les paramètres à estimer
	- définir les valeurs min et max pour chaque paramètre (ligne 11 et 12 du fichier)
	- définir les coefficients pour chaque variable vobs réduit (ligne 13 du fichier)
	- définir l'optimizer à utiliser : D = DE, G = RGenoud, A = Simulated Annealing (ligne 14 du fichier)
	- définir la méthode RSME à utiliser : RS = RSME-sum, REC = RMSE-EC, RC = RMSE-coef, RECC = RMSE-EC-coef (ligne 15 du fichier)
	- définir le nombre d'itération de l'algorithme (ligne 16 du fichier)
	- définir la tolérance à la solution pour la convergence de l'algorithme, ex : 0.05 = 5% (ligne 17 du fichier)

Lancer le script
Normalement ça marche :)

Plusieurs commandes peuvent être lancés dans la console R après avoir lancé le script :
	- res : résumé des résultats de l'algorithme
	- resPlot() : trace les graphiques pour chaque variable vobs, retourne également l'erreur (en %) pour chaque variable
	- dePlot() : graphiques spécifiques à l'algorithme DE
	- allPlot() : trace les graphiques de toutes les variables calculées (pas de points Vobs ici)
	- resOptim : détaille des résultats de l'algorithme

* Le fichier des écart-type sera de même forme que le fichier Vobs, pas besoin de préciser cette variable si la méthode choisie est RS ou RC

** Le vecteur des coef défini le coefficient pour chaque variable du vobs réduit (par ordre alphabétique), pas besoin de préciser cete variable si la méthode choisie est RS ou REC
